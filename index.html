Trốn Chướng Ngại Vật

Điều khiển nhân vật, né chướng ngại vật rơi — ghi điểm theo thời gian. Chơi ngay trên trình duyệt.

▶ Chơi Ngay   Hướng dẫn

🎯 Lối chơi

Điều khiển ← → để né. Điểm tăng theo giây. Nhiều loại chướng ngại: tĩnh, di chuyển, tường có khe.

⚙️ Tương thích

Chạy trên trình duyệt desktop và mobile (Chrome/Edge/Firefox). Không cần cài đặt.

📈 Chia sẻ

Chia link: người khác có thể chơi ngay. Sử dụng GitHub Pages (miễn phí).

Hướng dẫn nhanh

  1. Mở play.html để chơi (hoặc bấm "Chơi Ngay").
  2. Nếu muốn Google index nhanh: xác minh site (Search Console) — dùng meta tag hoặc upload file HTML do Google cung cấp.
  3. Để xác minh bằng file HTML, upload file google6b4bf1494a86926b.html vào gốc repo (file bạn có thể dùng chính xác như file mà Google đã cấp).

Lưu ý: nếu bạn dùng GitHub Pages và muốn link gọn, đổi play.html thành index.html hoặc đặt một index.html như file này.
